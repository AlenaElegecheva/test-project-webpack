## Обзор проекта

### Задачи проекта

Создать react-приложение на webpack с модальными окнами.

### Функциональность проекта

Функционально проект представляет из себя SPA, с последовательным открытием модальных окон.

### Директории проекта

- `src/components` — директория с компонентами
- `src/vendor` — директория с файлами библотек
- `src/images` — директория с изображениями

### Запуск проекта

- `npm run build` — запуск проекта в режиме продакшн
- `npm start` — запуск проекта в режиме разработки
